# Quantum Key Exchange Cracking Example

This project showcases a basic implementation of quantum computing concepts, specifically Shor's Algorithm, to demonstrate how quantum computers can be used to factorize large numbers efficiently. It serves as an example of applying quantum algorithms to real-world cryptography challenges.

## Setup

To run the quantum algorithms in this project, ensure you have the following prerequisites:

- Python 3.8 or higher
- Qiskit (Quantum Computing SDK for Python)
- Quantum simulator or access to a quantum computer through IBM Quantum Experience

### Required Packages
Python package install:
`pip install mpmath sympy`
